# Event-Driven E-Commerce Backend

A production-inspired, event-driven e-commerce backend built using
Java 17, Spring Boot microservices, Kafka, Redis, PostgreSQL, and JWT security.

## Project Goals
- Learn real-world backend engineering
- Practice system design concepts
- Build a resume-ready microservices project

## Architecture
- Microservices-based
- Event-driven communication (Kafka)
- Saga pattern for distributed transactions
